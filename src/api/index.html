HnS Login

How to use HnS Netai API:

// In order to use HnS Netai API you'll need to register you're website and request an API key. You'll then authenticate your website by calling the following function.

HNS.init({"apikey":"yourapikey"});

// You can track and control your website based on the status of being logged in or logged out by using the functions HNS.loggedIn and HNS.loggedOut.

HNS.loggedIn = function(){
	$("#in").show();
	$("#out").hide();
	log("performing login");
};
HNS.loggedOut = function(){
	$("#out").show();
	$("#in").hide();
	log("performing logout");
};

// You can manually call the login and logout functions by the HNS.login and HNS.logout functions.

$("#custom-login-link").click(function(){
	HNS.login();
});
$("#custom-logout-link").click(function(){
	HNS.logout();
});

// In order to include hns login and logout buttons you'll need to include a tag with an id of #hns-login-button or #hns-logout-button.

<div id="hns-login-button">HnS Login</div>
<div id="hns-logout-button">HnS Logout</div>

// You can also declare your button tag by using custom id's or classes as long as you pass them to the HNS.init function.

<div id="custom-login">HnS Login</div>
<div class="custom-logout">HnS Logout</div>

you pass these values to HNS.init like this (you don't have to include both the login and logout only the one's you want to use).

HNS.init({"apikey":"yourapikey","logintarget":"#custom-login","logouttarget":".custom-logout"});
